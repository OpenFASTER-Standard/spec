OpenFASTER defines an interoperable protocol and data exchange format that Certified Financial Intermediaries (CFIs) use to exchange the data and events required for indirect reporting under MiKaDiv and FASTER. The standard supports both pull and push modes, is based on BIRD (ECB), and specifies end-to-end encryption for all transmissions between CFIs.

This is a working draft. All details marked with TODO are placeholders for future specification work.

Introduction

OpenFASTER is a vendor-independent standard for data exchange in the EU withholding tax and dividend reporting domain. It was developed in cooperation with the financial industry to enable CFIs to interoperate for indirect reporting, including the collection of supplementary data such as (e)TRCs. The scope is purely operational: building technical infrastructure that makes indirect reporting efficient and standardized across the EU.
